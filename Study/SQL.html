Syntax

  1. 기본형태

    1. SELECT * FROM Customers;
    2. SELECT Column1, column2,... FROM table_name;

  2. 주요 SQL 명령문

    • SELECT - extracts data from a database 데이터베이스에서 데이터를 추출
    • UPDATE - updates data in a database 데이터베이스의 데이터를 업데이트
    • DELETE - deletes data from a database 데이터베이스에서 데이터를 삭제
    • INSERT INTO - inserts new data into a database 데이터베이스에 새 데이터를 삽입
    • CREATE DATABASE - creates a new database 새 데이터베이스를 만듬.
    • ALTER DATABASE - modifies a database 데이터베이스를 수정
    • CREATE TABLE - creates a new table 새 테이블을 만듬
    • ALTER TABLE - modifies a table 테이블 수정
    • DROP TABLE - deletes a table 테이블을 삭제
    • CREATE INDEX - creates an index (search key)색인 (검색 키) 작성
    • DROP INDEX - deletes an index 색인을 삭제

References

  • SELECT DISTINCT column1, column2, ... FROM table_name;
    SELECT DISTINCT 문은 구별되는 (다른) 값만 리턴하는 데 사용됨(중복값제외)
  • SELECT column1, column2, ... FROM table_name WHERE condition;
    WHERE 절은 레코드를 필터링하는 데 사용됨 WHERE 절은 지정된 조건을 충족하는 레코드 만 추출하는 데 사용됨
  • SELECT column1, column2, ... FROM table_name WHERE condition1 AND condition2 AND condition3 ...;
    WHERE 절은 AND, OR 및 NOT 연산자와 결합 될 수 있음 AND 및 OR 연산자는 둘 이상의 조건을 기반으로 레코드를 필터링하는 데 사용 조건이 참이 아닌 경우 NOT 연산자는 레코드를 표시
  • SELECT column1, column2, ... FROM table_name ORDER BY column1, column2, ... ASC|DESC;
    ORDER BY 키워드는 결과 집합을 오름차순 또는 내림차순으로 정렬하는 데 사용 ORDER BY는 기본적으로 오름차순으로 정렬. 내림차순으로 정렬하려면 DESC 사용
